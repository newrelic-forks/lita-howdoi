# lita-howdoi

Crawls Stack Overflow for the first answer it finds with a solution to some query.

## Installation

Add lita-howdoi to your Lita instance's Gemfile:

``` ruby
gem "lita-howdoi"
```

## Usage

```
@robot: howdoi split a string in ruby
...
```
